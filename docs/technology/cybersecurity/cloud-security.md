# Cloud Security

### Overview

Securing cloud infrastructure and services requires specialized approaches.

### Key Areas

#### Cloud Service Models
- **IaaS Security**:
  - Network segmentation
  - VM security
  - Storage encryption
- **PaaS Security**:
  - API security
  - Development pipeline security
  - Platform hardening
- **SaaS Security**:
  - Data privacy
  - Access control
  - Compliance monitoring

#### Container Security
- **Docker Security**:
  - Image scanning
  - Runtime protection
  - Network isolation
- **Kubernetes Security**:
  - Pod security policies
  - RBAC configuration
  - Network policies
